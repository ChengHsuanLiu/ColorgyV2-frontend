主揪介紹
完整服務幫大家訂書的你
什麼是主揪？

揪團負責人，負責收書、發書和為跟團的同學服務！

願意當主揪的人，可以獲得100元的折扣，Colorgy謝謝您為台灣整體訂書環境的提升做出貢獻。（主揪義務說明）

1. 成為主揪
點選開始揪團 ，按照指示，選出課程所需用書，成為當課的訂購書籍負責人，開起ㄧ波團購！
2. 選擇時間地點
決定此團的截止跟團時間、寄送宅即便的地點，並填寫你想以怎樣的方式讓跟團的同學們拿到書。
3. 完成開團
開團成功後，可以至主揪介面隨時瞭解團訂狀況（要是訂購人數沒有滿5人，則會流團喔～）
4. 發送領書訊息
領到書後，請至主揪介面傳送通知給同學，會同步更新同學的領書狀況！

什麼是主揪？

揪團負責人，負責收書、發書和為跟團的同學服務！

願意當主揪的人，可以獲得100元的折扣，Colorgy謝謝您為台灣整體訂書環境的提升做出貢獻。（主揪義務說明）

1. 放入購物書包
將要購買的教科書，加入購物書包 ( 記得要確認是否為正確的團喔 ! )
2. 前往結帳
進入購物書包，選擇付款方式和發票種類 ( 超商付款詳情說明 )
3. 完成訂購
送出訂單後，可以至訂單查詢隨時追蹤書籍況

什麼是主揪？

揪團負責人，負責收書、發書和為跟團的同學服務！

願意當主揪的人，可以獲得100元的折扣，Colorgy謝謝您為台灣整體訂書環境的提升做出貢獻。（主揪義務說明）

1. 主揪選定地點
你所訂購的書籍，將會以主揪決定的地點做為宅即便配送的目的地。確定團購成功後，書商將在3-5天內送出書籍！
2. 關注訂單查詢
當書到達主揪手中，訂單查詢將同步顯示書的狀況，也可在訂單查詢上看見主揪的發書方式！
3. 主揪發書
不管是系會領書、上課發書，或是其他的領書位置～主揪都會將書籍送至同學的手中！
